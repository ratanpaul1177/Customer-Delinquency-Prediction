# Customer Delinquency Prediction Using Machine Learning

## Project Overview

This project focuses on predicting customer delinquency risk using machine learning techniques. The model analyzes customer financial and behavioral data to identify customers who may have difficulty making payments.

## Objective

- Analyze customer financial information.
- Identify factors affecting delinquency.
- Build a machine learning model for prediction.
- Help financial institutions manage credit risk.

## Dataset Features

The dataset contains:

- Age
- Income
- Credit Score
- Credit Utilization
- Missed Payments
- Loan Balance
- Debt-to-Income Ratio
- Employment Status
- Account Tenure
- Credit Card Type
- Location

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Joblib

## Machine Learning Models

The following models were used:

1. Logistic Regression
2. Random Forest Classifier

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Preprocessing
5. Model Training
6. Model Evaluation
7. Feature Importance Analysis

## Results

The model successfully predicts customer delinquency risk and identifies important factors influencing repayment behavior.

## Conclusion

This project demonstrates how machine learning can support financial institutions in predicting customer risk and improving credit risk management.
