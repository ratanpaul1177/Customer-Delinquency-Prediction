# Customer Delinquency Prediction - EDA Project

## Project Overview
This project focuses on analyzing customer financial data to understand patterns related to delinquent accounts using Exploratory Data Analysis (EDA).

## Objective
The main objective is to identify factors that influence customer delinquency by analyzing credit behavior, financial status, and payment history.

## Dataset Information

The dataset contains customer details including:

- Customer ID
- Age
- Income
- Credit Score
- Credit Utilization
- Missed Payments
- Delinquent Account
- Loan Balance
- Debt-to-Income Ratio
- Employment Status
- Account Tenure
- Credit Card Type
- Location
- Monthly payment history (Month 1 - Month 6)

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Machine Learning (Model Building)

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Missing Value Analysis
5. Statistical Analysis
6. Data Visualization
7. Feature Analysis
8. Model Development

## Key Analysis Performed

- Distribution analysis of customer features
- Credit score analysis
- Income and loan balance analysis
- Payment behavior analysis
- Correlation analysis
- Identification of delinquency patterns
